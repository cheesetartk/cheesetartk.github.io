## 拉取镜像

```
docker pull halcyonazure/lsky-pro-docker
```
## 运行容器
```
mkdir /usr/local/share/applications/lskypro
```
```
docker run -it -d --name lsky-pro -p 8089:8089 -v /usr/local/share/applications/lskypro:/var/www/html --restart=unless-stopped halcyonazure/lsky-pro-docker
```

## 安装配置
```
10.24.0.2:8089
```
