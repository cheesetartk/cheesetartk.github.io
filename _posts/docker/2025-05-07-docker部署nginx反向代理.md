tags: docker

```
mkdir /usr/local/share/applications/nginx
mkdir /usr/local/share/applications/nginx/html
mkdir /usr/local/share/applications/nginx/conf
mkdir /usr/local/share/applications/nginx/logs
```
先创建简单容器复制配置文件
```
docker run --name nginx -p 80:80 -d nginx
docker cp nginx:/etc/nginx/nginx.conf /usr/local/share/applications/nginx/conf/nginx.conf
docker cp nginx:/etc/nginx/conf.d /usr/local/share/applications/nginx/conf/conf.d
docker cp nginx:/usr/share/nginx/html /usr/local/share/applications/nginx
docker rm -f nginx
```
编辑配置文件
```
新建mime.types保存至/usr/local/share/applications/nginx/conf/conf.d目录下
编辑/usr/local/share/applications/nginx/conf/nginx.conf更改路径include   /etc/nginx/conf.d/mime.types;
```

启动容器
```
docker run --name nginx-a -p 8680:80 -v /usr/local/share/applications/nginx/html:/usr/share/nginx/html -v /usr/local/share/applications/nginx/conf:/etc/nginx -v /usr/local/share/applications/nginx/logs:/var/log/nginx --privileged=true -d --restart=unless-stopped nginx 
```
