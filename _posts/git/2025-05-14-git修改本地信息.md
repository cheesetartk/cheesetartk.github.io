tags: git

缓存输入的用户名和密码：

git config --global credential.helper wincred



清除掉缓存在git中的用户名和密码

git credential-manager uninstall



重新生成ssh-key
rm -rf ~/.ssh


重新设置自己的邮箱和用户名
查看用户名和邮箱地址：

git config user.name

git config user.email



修改用户名和邮箱地址：

git config --global user.name "username"

git config --global user.email "email"
