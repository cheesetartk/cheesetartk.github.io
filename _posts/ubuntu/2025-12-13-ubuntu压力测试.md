#### ubuntu GPU压力测试

```bash
先安装NVIDIA驱动
apt install g++
apt install nvidia-cuda-toolkit
git clone https://github.com/wilicc/gpu-burn.git
cd gpu-burn
make
```

编译完成后执行测试：

```bash
./gpu_burn 3600  # 持续3600秒（1小时），默认使用所有GPU
tail -f gpu_burn.log 可以查看烧卡日志
```

支持指定GPU卡：

```bash
CUDA_VISIBLE_DEVICES=0 ./gpu_burn 600  # 仅测试0号GPU，持续10分钟
```

#### ubuntu CPU压力测试

```bash
apt-get install stress
stress --cpu 4 --timeout 60
```

这个命令会使用4个CPU核心进行测试，持续60秒。



#### ubuntu 内存压力测试

stress --vm 2 --vm-bytes 1G --vm-keep --timeout 60

测试2个进程,每个进程分配1G(M)内存,持续60秒



### 自动获取cpu数量压测CPU：

stress-ng --cpu $(nproc) --timeout 1800s --metrics-brief

### 自动获取内存数量压测：

stress-ng --vm 4 --vm-bytes 90% --timeout 360000s --metrics-brief
