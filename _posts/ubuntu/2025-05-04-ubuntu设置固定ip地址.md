---
title: "ubuntu设置固定ip地址"
tags: ubuntu
---



#### 找到 Netplan 的配置文件

```
ip a
cd /etc/netplan
ls -l
sudo vi 50-cloud-init.yaml
```
#### 配置IP地址

```
network:
    version: 2
    ethernets:
        ens33:
            addresses: [192.168.100.122/24] 
            gateway4: 网关IP地址
			nameservers:
  				addresses: [DNS服务器IP地址]

```
#### 保存配置

```
sudo netplan apply
```
