手动安装指定驱动

sudo apt-get update

ubuntu-drivers devices 

sudo apt install nvidia-390(对应版本)



自动安装所有驱动

sudo apt update
sudo apt install ubuntu-drivers-common
sudo ubuntu-drivers autoinstall



自动安装所有驱动并重启

apt update && apt -y upgrade && apt autoremove && apt -y install build-essential && apt install ubuntu-drivers-common && ubuntu-drivers autoinstall && reboot
